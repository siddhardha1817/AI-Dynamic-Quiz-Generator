# AI-Dynamic-Quiz-Generator
An AI-powered web application built using Python and Flask that supports user authentication, quiz generation, score tracking, and AI-based response handling.
---
# Features
* AI-Powered Quiz Generation
* Score Tracking
* Interactive Web Interface
* Flask Backend Integration
* Secure Session Handling
* Responsive Frontend
* GitHub Integration Support

---

# Tech Stack

## Frontend

* HTML
* CSS
* JavaScript

## Backend

* Python
* Flask

## AI Integration

* OpenAI / BoltIOT AI API

# Installation

## Clone the Repository

```bash
git clone https://github.com/siddhardha1817/AI-Dynamic-Quiz-Generator.git
```

## Move into the Project Directory

```bash
cd AI-Dynamic-Quiz-Generator
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

# Environment Variables

Create a `.env` file or add secrets in Replit.

Example:

```env
OPENAI_API_KEY=your_api_key_here
```

---

# Run the Application

```bash
python main.py
```
